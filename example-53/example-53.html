CSS Tutorial

What is CSS?

CSS (cascading style sheets) is used to define the style of your webpage.

CSS allows you to create rules that specify how the content of each element should appear.

What element do we use for external css files?

The link element

The style element is used for internal elements

The Link and Style elements are reside inside the the head elements

The value of the type attribute for the style element should be "text/css"
